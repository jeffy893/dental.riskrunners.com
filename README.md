# Risk Runners Dental Captive

A feasibility study rendered in code for a captive insurance syndicate serving 12 dentists. Built as a static site for GitHub Pages with Python-powered Monte Carlo simulations.

**Live site:** [dental.riskrunners.com](https://dental.riskrunners.com)

## What Is This?

Commercial malpractice and liability insurance is a sunk cost — dentists pay premiums year after year, and if there are no claims, the insurer keeps the profit. This project models an alternative: 12 dentists form a syndicate, pool risk through their own captive insurance entity, smooth cash flow, and retain wealth collectively.

The captive covers **fortuitous risk only** — equipment failures, advanced procedure complications, cybersecurity breaches. Speculative risk (market changes, intentional acts) is expressly excluded.

## Running the Simulations

Requires Python 3.10+.

```bash
# Option 1: Use the runner script
./run_simulation.sh

# Option 2: Manual
python3.10 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cd src
python main.py
```

This generates:
- `docs/assets/monte_carlo_results.png` — solvency analysis charts
- `docs/assets/claims_analysis.png` — claims distribution by risk type and specialty
- `docs/data/simulatedPolicies.json` — policy data for the interactive policy slips page
- `data/*.csv` — raw simulation data

## Simulation Details

### Monte Carlo Solvency (risk_model.py)
- 5,000 independent simulations over a 10-year horizon
- $1M initial capitalization, 12 dentists, ~$50K base premium each
- Claim frequency: per-dentist risk probability based on skill, tech adoption, and experience
- Claim severity: lognormal distributions by risk type (equipment $35K±15K, procedures $20K±8K, cyber $45K±20K)
- 4% annual investment return on reserves, 10% operating expense load

### Claims Analysis (main.py)
- 10-year claims simulation across all 12 dentists
- Breakdown by risk type (equipment failure 40%, procedure complications 35%, cyber breach 25%)
- Distribution across 4 dental specialties (General, Implant, Cosmetic, Pediatric)

### Policy Generation (main.py)
- Generates simulated Lloyd's MRC format policies for each dentist
- Experience-modified premiums, claims history, reinsurance structure
- Consumed by the policy-slips.html page via fetch()
